# Budget Base

Minimal React scaffold for a local-first budgeting app.

## Structure

- `src/App.jsx`: app composition and state wiring
- `src/components/`: compact UI sections
- `src/data/`: sample state and storage keys
- `src/lib/`: budgeting math and formatting helpers

## Run

1. `npm install`
2. `npm run dev`

## Current focus

- local monthly budgeting
- planned vs actual tracking
- compact banking/spreadsheet-style UI
- persisted dark mode
- JSON import and export
